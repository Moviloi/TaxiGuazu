- `TASK_PLAN.md` — Plan de tarea (artefacto de misión activa)
- `SYSTEM_STATE.md` — Estado del sistema (artefacto de misión activa)
- `SYSTEM_STATE_AUDIT.md` — Auditoría de estado (artefacto de misión activa)
- `DESIGN_SPEC.md` — Especificación de diseño (artefacto de misión activa)
- `DECISION_RECORD.md` — Registro de decisiones (artefacto de misión activa)
- `PATTERN_EXTRACTION.md` — Extracción de patrones (artefacto de misión activa)
- `VALIDATION_REPORT.md` — Reporte de validación (artefacto de misión activa)
- `DIAGNOSTIC_REPORT.md` — Reporte de diagnóstico (artefacto de misión activa)
